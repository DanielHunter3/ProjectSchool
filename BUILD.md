Руководство по сборке и запуске файлов в проекте
-------------------------------------------------
Заходите в папки ***boost*** и ***alloc***, запускайте в них build.sh через:
```bash
./build.sh
```
Но могут появиться проблемы с правами доступа. В этом случае пишем перед запуском:
```bash
chmod +x build.sh
```
